Java GUI Game - Tree Decorating

Problem Statement & Project Scope

Skills: Java, GUI

This is a game that aims to help kids improve their color recognition and counting skills when practicing mouse using.
The tree is a Polygon that can be decorated with ornaments using mouse clicks. There is a counter of the ornaments. The buttons labeled Start and Stop implement a light show. When “Start” is pressed the lights blink. When “Stop” is pressed the blinking stops. (New ornaments can be added during a light show.)
I created two separate classes. The first, Ornament.java, defines an ornament class. Each ornament will be the same shape, but the color and placement can vary. The second, TreePanel.java defines a decorated Holiday tree.

Java GUI Game - Catch the Santa!

Problem Statement & Project Scope

Skills: Java, GUI

This is a game that aims to help kids improve their reaction time and focusing skills when practicing mouse using.
The Santa moves every second and be visible at 75% of the time and invisible for 25% of the time. If the player clicks on the mole before it disappears, it counts as a hit. The number of hits and misses displays at the top of the panel in black. You can hit a mole multiple times before it disappears.
